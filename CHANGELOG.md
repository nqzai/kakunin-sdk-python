# Changelog

All notable changes to this project are documented here. The format is based on
[Keep a Changelog](https://keepachangelog.com/en/1.1.0/) and this project adheres
to [Semantic Versioning](https://semver.org/).

## [0.2.0] — 2026-07-07
### Added
- Public release of `kakunin` (Apache-2.0) on PyPI via OIDC trusted publishing.
- Async client plus integrations for LangChain, LlamaIndex, CrewAI, AutoGen,
  LangGraph, CAMEL-AI, and OpenAI Assistants; `verify_agent_scope` decorator.
### Changed
- Requires Python 3.10+.
